        Data description parameters of the config:
        

        \begin{itemize}
        \item[{\bf allowed chars:}] abcdefghijklmnopqrstuvwxyzäöüß 
        \item[{\bf number of targets:}] 2
        \item[{\bf number of character classes:}] 32 (one more than char count for the generic class)

        \end{itemize}
        

        
        Network description parameters of the config:
        

        \begin{itemize}
        \item[{\bf n syllables:}] 100 number of patterns in first layer, which is a combination of some characters, i.e., something like a syllable
        \item[{\bf syllable length:}] 4 number of characters in 'syllable'
        \item[{\bf n words:}] 100 number of 'word' patterns which are combined 'syllables'
        \item[{\bf word length:}] 3 number of 'syllables' in each 'word' pattern
        \item[{\bf output number:}] 3 dimension of fully connected pre-output layer
        \item[{\bf strides 1:}] 1 strides in the first layer along the 'sentence'
        \item[{\bf strides 2:}] 1 strides in the second layer along the 'syllables'
        \end{itemize}
